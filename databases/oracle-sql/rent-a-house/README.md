# 🏠 Rent A House Database

Proyecto de base de datos desarrollado en SQL como práctica de modelado relacional y consultas.

El sistema representa un escenario simplificado para la gestión de clientes, propiedades inmobiliarias y convenios entre inmobiliarias.

---

## Características

- Creación de tablas.
- Relaciones mediante claves foráneas.
- Inserción de datos.
- Consultas SQL.
- Vistas.
- Actualización de registros.

---

## Tecnologías

- Oracle SQL

---

## Modelo de datos

El proyecto incluye las siguientes entidades:

- Clientes
- Propiedades
- Inmobiliarias
- Convenios
