# 👥 Employee Management System

Aplicación web desarrollada con **HTML, CSS y JavaScript** como proyecto académico.

El sistema permite gestionar información de personal mediante una interfaz web sencilla y organizada.

---

## Características

- Gestión de personal.
- Navegación entre páginas.
- Interfaz desarrollada con HTML y CSS.
- Funcionalidades implementadas con JavaScript.
- Diseño personalizado.

---

## Tecnologías

- HTML5
- CSS3
- JavaScript

---

## Estructura

```text
employee-management-system/
├── index.html
├── gestionPersonal.html
├── script.js
├── styles.css
└── assets/
```

---

## Ejecución

Abrir:

```text
index.html
```

en cualquier navegador.

---

## Objetivos de aprendizaje

- Manipulación del DOM.
- Eventos.
- Formularios.
- Organización de un proyecto web.
- JavaScript Vanilla.
