# 🖼️ Web Galería

Sitio web desarrollado como proyecto de práctica utilizando **HTML, CSS y JavaScript**.

El objetivo principal fue aprender la estructura de un sitio web multipágina, aplicar estilos personalizados y agregar interactividad mediante JavaScript.

---

## ✨ Características

- Sitio web multipágina.
- Página principal.
- Galería de imágenes.
- Página de contacto.
- Estilos personalizados con CSS.
- Funcionalidades implementadas con JavaScript.

---

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript

---

## 📂 Estructura

```text
web-galeria/
│
├── index.html
├── galeria.html
├── contacto.html
├── styles.css
├── index.js
├── galeria.js
├── contacto.js
└── assets/
```

---

## 🚀 Cómo ejecutar

No requiere instalación.

Simplemente abre:

```text
index.html
```

en cualquier navegador moderno.

---

## 📚 Objetivos de aprendizaje

Durante este proyecto practiqué:

- HTML semántico.
- CSS.
- Diseño multipágina.
- JavaScript.
- Organización de archivos.
- Navegación entre páginas.
