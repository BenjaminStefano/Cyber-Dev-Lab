# 🌙 Club Nocturno

Programa de consola desarrollado con **Python** que determina si una persona puede ingresar a un club nocturno según su edad.

Este proyecto fue creado durante mi aprendizaje de los fundamentos de Python para practicar la entrada de datos, la conversión de tipos y las estructuras condicionales.

---

## 📸 Características

- Solicita la edad del usuario.
- Valida que la edad sea coherente.
- Determina si el usuario puede ingresar al club.
- Muestra diferentes mensajes según la edad ingresada.
