# 🔐 Login Tkinter

Aplicación de escritorio desarrollada con **Python** y **Tkinter** que simula un sistema básico de autenticación mediante usuario y contraseña.

Este proyecto fue creado para practicar el desarrollo de interfaces gráficas, la captura de datos desde formularios y la validación de información ingresada por el usuario.

---

## 📸 Características

- Interfaz gráfica desarrollada con Tkinter.
- Campo de usuario.
- Campo de contraseña.
- Validación de credenciales.
- Mensajes de acceso permitido o denegado.

---

## 🛠️ Tecnologías

- Python 3
- Tkinter
