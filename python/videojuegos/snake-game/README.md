# 🐍 Snake Game

Un clásico juego de Snake desarrollado con **Python** y **Pygame** como proyecto de aprendizaje.

Este proyecto fue creado con el objetivo de practicar conceptos fundamentales del desarrollo de videojuegos, como el manejo de eventos, bucles de juego, colisiones, renderizado en pantalla y control del estado del juego.

---

## 📸 Características

- Movimiento mediante el teclado.
- Sistema de puntuación.
- Generación aleatoria de comida.
- Detección de colisiones.
- Reinicio del juego después de perder.
- Interfaz desarrollada con Pygame.

---

## 🛠️ Tecnologías

- Python 3
- Pygame

## 🚀 Instalación

Clona el repositorio:

```bash
git clone https://github.com/BenjaminStefano/Cyber-Dev-Lab.git
```

Entra al proyecto:

```bash
cd python/videojuegos/snake-game
```

Instala las dependencias:

```bash
pip install -r requirements.txt
```

Ejecuta el juego:

```bash
python main.py
```
