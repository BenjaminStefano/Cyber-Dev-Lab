# Security Policy

## Responsible Use

Las herramientas y proyectos de este repositorio están destinados exclusivamente a:

- Aprendizaje
- Investigación
- Laboratorios controlados
- Auditorías autorizadas

El uso de estas herramientas contra sistemas sin autorización es responsabilidad exclusiva del usuario.

## Reporte de vulnerabilidades

Si encuentras una vulnerabilidad o problema de seguridad en alguno de los proyectos, puedes abrir un Issue o ponerte en contacto conmigo mediante correo electrónico.
