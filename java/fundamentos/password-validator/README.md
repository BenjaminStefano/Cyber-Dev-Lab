# ☕ Password Validator

> First Java project of **Cyber-Dev-Lab**

A simple command-line application developed in Java that evaluates the strength of a password based on common security rules.

This project was created to practice Java fundamentals while learning the language.

---

## Features

- Minimum length validation
- Uppercase detection
- Lowercase detection
- Number detection
- Special character detection
- Space validation
- Password strength score

---

- Java
