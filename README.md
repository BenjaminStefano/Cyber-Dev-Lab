# 🚀 Cyber-Dev-Lab

Bienvenido a mi repositorio.

Este espacio reúne los proyectos que desarrollo mientras avanzo en mi formación como programador y futuro profesional en Ciberseguridad.

Aquí encontrarás desde pequeños ejercicios para aprender nuevos lenguajes hasta herramientas más completas enfocadas en automatización, desarrollo web, auditorías de seguridad y análisis de sistemas.

Mi objetivo es documentar mi progreso y compartir cada etapa de mi aprendizaje mediante proyectos reales.


# 📌 Áreas del repositorio

## 🐍 Python

Proyectos de automatización, scripts, interfaces gráficas, herramientas para auditorías, análisis de datos y utilidades.

Ejemplos:

- Automatización de tareas
- Herramientas OSINT
- Escáneres de red
- Scripts de reconocimiento
- Utilidades para pentesting
- Aplicaciones con Tkinter



## ☕ Java

Proyectos orientados a fortalecer la programación orientada a objetos.

Incluye:

- Consola
- Interfaces gráficas
- Estructuras de datos
- Algoritmos
- Buenas prácticas



## 🌐 Desarrollo Web

Aplicaciones y sitios web desarrollados con distintas tecnologías.

Tecnologías que iré utilizando:

- HTML
- CSS
- JavaScript
- PHP
- WordPress
- Frameworks (Angular, Bootstrap, NodeJS, etc)



## 🔐 Ciberseguridad

Esta será la sección principal del repositorio.

Aquí desarrollaré herramientas para laboratorios y entornos autorizados con fines educativos.

Entre ellas:

- Escáneres de puertos
- Enumeración de servicios
- OSINT
- Automatización de auditorías
- Análisis de vulnerabilidades
- Scripts para Linux
- Herramientas para Parrot OS
- Utilidades para Capture The Flag (CTF)

**Todo el contenido tiene fines educativos y de aprendizaje.**



# 🎯 Objetivo

Construir un portafolio que refleje mi crecimiento como desarrollador y especialista en Ciberseguridad, documentando proyectos cada vez más completos y desafiantes, aprendiendo de los posibles errores y obstaculos que aparezcan en el camino.
